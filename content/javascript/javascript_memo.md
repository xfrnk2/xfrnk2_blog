#### ES6은 무엇인가?
+ 화살표 함수를 검색하다가 여기까지 오게되었다.
+ 바스크립트는 ECMAScript + BOM(Browser Object Model) + DOM(Document Object Model)라는 1개의 코어와 2개의 모델로 이루어져 있다.
+ this를 동적으로 바인딩하지 않는 화살표 함수의 문제점이 해결되었다.
